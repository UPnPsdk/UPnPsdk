UPNPLIB_EXTERN SSL_CTX* gSslCtx

Pointer to the global available SSL Context object if OpenSSL is enabled.

If OpenSSL is available and configured to use then an SSL_CTX object is initialized and the pointer to it is global available so it can be used for SSL connections.
