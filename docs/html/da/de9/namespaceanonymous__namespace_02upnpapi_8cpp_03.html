int anonymous_namespace{upnpapi.cpp}::GetIfInfo ( const char *  a_IfName)

Retrieve network interface information for a given interface name and keep it in global variables.

If nullptr, we'll find the first suitable network interface for operation. We'll retrieve the following information from the interface:

  • gIF_NAME -> Interface name (by input or found).
  • gIF_IPV4 -> IPv4 address (if any).
  • gIF_IPV6 -> IPv6 address (if any).
  • gIF_IPV6_ULA_GUA -> ULA or GUA IPv6 address (if any)
  • gIF_INDEX -> Interface index number.
Returns
UPNP_E_SUCCESS on success.
