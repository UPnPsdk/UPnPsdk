Detailed Description

Trivial ::sockaddr structures enhanced with methods.

// Usage e.g.:
::sockaddr_storage saddr{};
SSockaddr saObj;
::memcpy(&saObj.ss, &saddr, saObj.sizeof_ss());
std::cout << "netaddress of saObj is " << saObj.netaddr() << "\n";

false otherwise
