Detailed Description

Trivial ::sockaddr structures enhanced with methods.

~$ // Usage e.g.:
~$ ::sockaddr_storage saddr{};
~$ SSockaddr saObj;
~$ ::memcpy(&saObj.ss, &saddr, saObj.sizeof_ss());
~$ std::cout << "netaddress of saObj is " << saObj.get_netaddr() << "\n";

This structure should be usable on a low level like the trival C struct ::sockaddr_storage but provides additional methods to manage its data.

Set socket address from a netaddress.
