Detailed Description

This manage a URL and is based on the URL Living Standard

At time the Url parser was coded this snapshot was used:
Commit Snapshot - Last Updated 21 February 2022 of the URL standard.
To understand the parser follow at that snapshot the basic URL parser.

This definition is helpful for coding. It is only understandable if you examine the code but I will have it available here.
"A URI is opaque if, and only if, it is absolute and its scheme-specific part does not begin with a slash character ('/'). An opaque URI has a scheme, a scheme-specific part, and possibly a fragment; all other components are undefined."
Example of an opaque url: mailto:a@b
REF: http://docs.oracle.com/javase/8/docs/api/java/net/URI.html#isOpaque&ndash;

To manual verify URLs conforming to the standard you can use the Live URL Viewer.
