Check for a numeric netaddress and return its address family.

Checks if a string represents a numeric IPv6 or IPv4 address without port and determines its address family.

Returns
On success: Address family AF_INET6 or AF_INET the address belongs to
On error: AF_UNSPEC, the address is alphanumeric (maybe a DNS name?)
